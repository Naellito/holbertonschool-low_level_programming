# C - argc, argv

## Files
### 0. It ain't what they call you, it's what you answer to
- **File:** `0-whatsmyname.c`
- **Description:** A program that prints its name, followed by a new line.

### 1. Silence is argument carried out by other means
- **File:** `1-args.c`
- **Description:** A program that prints the number of arguments passed into it.

### 2. The best argument against democracy is a five-minute conversation with the average voter
- **File:** `2-args.c`
- **Description:** A program that prints all arguments it receives, each on a new line.

### 3. Neither irony nor sarcasm is argument
- **File:** `3-mul.c`
- **Description:** A program that multiplies two numbers. If the program does not receive exactly two arguments, it prints "Error" and returns 1.

### 4. To infinity and beyond
- **File:** `4-add.c`
- **Description:** A program that adds positive numbers passed as arguments. If no number is passed, it prints 0. If any argument contains non-digit symbols, it prints "Error" and returns 1.
