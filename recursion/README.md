# Recursion Projects

This repository contains a series of C programming exercises focused on recursion. Each project demonstrates a different recursive function to solve common problems.

## Files

### 0-puts_recursion.c
A function that prints a string, followed by a new line, using recursion.

### 1-print_rev_recursion.c
A function that prints a string in reverse, using recursion.

### 2-strlen_recursion.c
A function that returns the length of a string, using recursion.

### 3-factorial.c
A function that returns the factorial of a given number. Returns -1 if the number is negative.

### 4-pow_recursion.c
A function that returns the value of `x` raised to the power of `y`. Returns -1 if `y` is negative.

### 5-sqrt_recursion.c
A function that returns the natural square root of a number. Returns -1 if the number does not have a natural square root.

### 6-is_prime_number.c
A function that checks if a number is a prime number. Returns 1 if the number is prime, otherwise 0.
